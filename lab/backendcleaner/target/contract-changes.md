# 契约变更日志（驱动覆盖后的前端 / testend 兼容）

> 重写时每改一处对外契约（REST envelope / 路由 / SSE 事件 / error code），在此**追加一条**。
> 覆盖回 `backend/` 后，前端 + testend 照此表逐条施工——这是兼容工作的**唯一施工图**，记糙了兼容就会乱。
