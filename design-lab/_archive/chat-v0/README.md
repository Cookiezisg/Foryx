# chat-v0 —— 冻结参考（不是活海洋，勿当 ocean 读）

这是 Chat 海洋的 **v0 快照**：一版「视觉参考 Claude Code」的形态草图。
它定下了对话海洋的**视觉语言**（海与岛 / 工具块折叠 / 右岛实体卡滑入 / composer / spring 动效），
但**内容是通用 Claude Code**（建 PR、git diff、Auto 模型选择器、Ultracode、`read_agent`/`update_agent`），
**与 Forgify 真实产品能力错位**。

v1 起把这套视觉语言**适配到 Forgify 真实对话引擎**（六型块 / 锻造四实体 / 人在环危险闸 / durable flowrun / progress 块 / todo），
活文件在 `oceans/chat/`。本目录仅作**视觉参考**保留（`../../shared` 相对路径仍可直接预览）。
