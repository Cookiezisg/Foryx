## 1. 这块做什么

对话的创建、列表展示、资产绑定管理。用户在这里找到所有对话，并进入具体对话。

---

## 2. 对话列表（对话 Tab 内）

### 2.1 整体结构

```
┌─────────────────────────┐
│  🔍 搜索对话             │
├─────────────────────────┤
│  + 新建对话              │
├─────────────────────────┤
│  · 供应商报价处理         │  ← 独立对话
│  · 邮件自动化探索  ⚡报价 │  ← 绑定了工作流（带徽章）
│  · 发送邮件工具锻造 📦   │  ← 绑定了工具
│  · 昨天的随便聊聊         │  ← 独立对话
│     ...                 │
└─────────────────────────┘
```

所有对话统一按最后活跃时间倒序排列，**不按绑定关系分组**。绑定了资产的对话在名称右侧显示小徽章（工作流用 ⚡，工具用 📦，加资产名称前 6 字）。

### 2.2 对话条目

每条对话显示：
- 对话名称
- 最后活跃时间（"3 分钟前" / "昨天" / "4 月 15 日"）
- 资产徽章（若绑定）
- 悬停时出现操作菜单（...）

### 2.3 操作菜单（右上角 ... 或右键）

- 重命名
- 调整绑定（打开资产选择器，可重新绑定或解除绑定）
- ── 分隔线 ──
- 归档（不删除，隐藏到归档区）
- 删除（二次确认）

---

## 3. 新建对话

- 点击"+ 新建对话"：立即创建，进入空白对话页，无资产绑定
- 对话名称自动设为"新对话"，AI 首次完整回复后，后台静默用轻量模型生成标题（最多 15 字），自动更新

---

## 4. 搜索

- 搜索框实时过滤对话列表（按名称匹配）
- 全文搜索（含消息内容）在数据量大时异步触发
- 清除搜索后恢复完整列表

---

## 5. 对话-资产绑定

### 5.1 绑定的产生

1. **对话中创建资产**：AI 在对话中创建了工具或工作流后自动绑定。若同一对话中创建了多个资产，以最后一个为准。
2. **从资产打开**：在资产页的 mini-sidebar 点击"新建对话"，对话创建时即绑定该资产。

### 5.2 绑定指示条

进入已绑定资产的对话时，对话视图顶部出现：

```
┌────────────────────────────────────────────────────────┐
│  📎 已关联到：报价处理工作流  [切换 ▼]  [解除关联]       │
└────────────────────────────────────────────────────────┘
```

- 点击"切换"：弹出资产选择器（列出所有工作流和工具），选择后更新绑定
- 点击"解除关联"：直接解除绑定，对话变为独立对话（无确认框，可立即重新绑定）
- 指示条始终显示在对话顶部，不可手动关闭（但解除绑定后自动消失）

### 5.3 绑定变化时的通知

- **自动绑定触发时**：对话顶部指示条从无到有出现（带一个短暂的高亮动画，提示用户注意）
- 不弹对话框，不打断当前对话

### 5.4 资产选择器

点击"切换"后出现的浮层：

```
┌────────────────────────────────┐
│ 选择关联资产           [关闭]  │
│ ──────────────────────────── │
│ 工作流                         │
│   ⚡ 报价处理流程               │
│   ⚡ 日报发送                   │
│ 工具                           │
│   📦 发送邮件                   │
│   📦 解析Excel                  │
└────────────────────────────────┘
```

---

## 6. 归档

- 归档的对话不出现在主列表
- 列表最底部有"查看归档"入口，点击展开所有归档对话
- 归档对话可恢复（重新出现在主列表）或永久删除

---

## 7. 验收标准

- [ ] 新建对话，AI 回复后自动命名（≤15 字）
- [ ] 对话列表按最后活跃时间倒序排列
- [ ] 绑定了资产的对话显示对应徽章
- [ ] 操作菜单：重命名即时生效，归档后消失，删除需二次确认
- [ ] 自动绑定：对话中创建工具/工作流后，顶部指示条出现
- [ ] 手动切换绑定：资产选择器列出所有资产，选择后指示条更新
- [ ] 解除绑定：指示条消失，列表徽章消失
- [ ] 搜索实时过滤名称，清空后恢复
- [ ] 归档对话从主列表消失，归档区可恢复
