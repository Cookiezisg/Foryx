## 1. 这块做什么

工具库是 Forgify 的"仓库"，用户可以查看、搜索、管理所有已锻造的工具。
工具是 Python 函数，可以在工作流中作为节点调用。

## 3. 工具卡片信息

每张工具卡片显示：
- 工具名称（粗体）
- 所属分类（小字）
- 状态标签：
  - `草稿`（灰色）— 创建了但未测试
  - `已测试`（绿色）— 最近一次测试通过
  - `测试失败`（红色）— 最近一次测试失败
- 最后测试时间（相对时间）
- 操作按钮：`详情`、`删除`

---

## 4. 操作

### 4.1 锻造新工具

点击"+ 锻造新工具" → 在当前对话或新建对话中开始工具锻造（跳转到 D2）

### 4.2 搜索

搜索框输入关键词 → 即时过滤工具名称和描述

### 4.3 分类筛选

分类来自工具的 `category` 字段，固定为：
- 全部（默认）
- 邮件
- 数据处理
- 网络
- 文件
- 其他

### 4.4 删除

点击删除 → 弹出确认："删除后，使用该工具的工作流将无法运行。确认删除 [工具名]？"  
确认后删除，工具库即时刷新。
