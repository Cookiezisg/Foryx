---

## 1. 这块做什么

用户打开 Forgify 看到的第一个界面。包括：
- 主窗口和整体布局框架
- 左侧导航栏（四个主入口 + 资产选中时的 mini-sidebar）
- 系统托盘 + 后台常驻行为
- 各 Tab 的空白占位状态

这一块不包含任何业务功能，只是整个产品的"骨架"。

---

## 2. 主窗口

### 2.1 尺寸与行为

- **默认尺寸**：1280 × 800
- **最小尺寸**：960 × 640（更小则内容无法正常显示）
- **可调整大小**：是
- **标题栏**：显示 "Forgify"，macOS 使用原生交通灯按钮，Windows 使用原生最小化/最大化/关闭

### 2.2 关闭行为

- 点击关闭按钮 → **隐藏窗口，不退出应用**（后台任务继续运行）
- 真正退出需从系统托盘菜单选择"退出 Forgify"
- 首次关闭时弹一次提示："Forgify 已最小化到系统托盘，后台任务继续运行。"（之后不再提示）

## 3. 整体布局

```
┌───────────────────────────────────────────────────────────┐
│  标题栏（原生）                                             │
├──────┬────────────┬──────────────────────────────────────┤
│      │            │                                      │
│  左  │  资产时     │                                      │
│  侧  │  出现的     │          内容区（各 Tab 的页面）       │
│  导  │  mini-     │                                      │
│  航  │  sidebar   │                                      │
│  栏  │  (200px)   │                                      │
│      │            │                                      │
└──────┴────────────┴──────────────────────────────────────┘
```

mini-sidebar 只在**资产 Tab 且选中了某个工作流或工具**时出现，其他时候不存在。

### 3.1 左侧导航栏

- **宽度**：64px（收起）/ 220px（展开，显示文字标签）
- **默认状态**：展开（首次使用），记忆用户上次状态
- 顶部：Forgify Logo / 图标
- 中间（主导航，从上到下）：
  - **Home**（图标：房子）
  - **对话**（图标：气泡）
  - **资产**（图标：包裹）— 包含工作流和工具
  - **Inbox**（图标：收件箱）— 有未读时显示红点角标
- 底部（次要操作）：
  - **设置**（图标：齿轮）
- 当前激活的 Tab 有高亮指示（左侧竖条 + 背景色）
- 收起/展开：点击顶部 Logo 区域的箭头按钮切换

### 3.2 资产 mini-sidebar

选中工作流或工具后，在主侧边栏右侧紧贴出现一个 200px 宽的 mini-sidebar：
- 顶部：所选资产的名称
- `[+ 新建对话]` 按钮
- 与该资产关联的对话列表（点击进入对话）

点击主侧边栏其他 Tab 或其他空白处，mini-sidebar 收起。

### 3.3 内容区

- 占据窗口剩余所有宽度（主侧边栏 + mini-sidebar 之后的部分）
- 各 Tab 保持各自的滚动位置和状态（不因切换 Tab 而重置）
- Tab 切换时内容区平滑过渡（简单淡入即可）

---

## 4. 系统托盘

- **图标**：Forgify 图标（支持 macOS light/dark 自适应）
- **左键单击**：显示/隐藏主窗口（toggle）
- **右键菜单**：
  - 显示 Forgify（当窗口隐藏时）
  - ── 分隔线 ──
  - 今日已运行 X 个工作流（只读状态行）
  - ── 分隔线 ──
  - 退出 Forgify

## 5. 各 Tab 的空白占位状态

这一切片里，四个 Tab 内容区都是空白占位，等后续切片填充。

| Tab | 占位显示 |
|---|---|
| Home | 空白，灰色文字："你的工作台，正在初始化..." |

| 资产 | 空白，灰色文字："你的工具和工作流会显示在这里" |
| Inbox | 空白，灰色文字："收件箱是空的" |
